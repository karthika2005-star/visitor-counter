Components Required
Arduino Uno
2 IR Sensors
16x2 LCD Display
Breadboard
Jumper Wires
Resistors
Power Supply



🔌 Working Principle
IR Sensor 1 detects when a person enters the room and increments the visitor count.
IR Sensor 2 detects when a person exits the room and decrements the count.
The Arduino processes the signals from the sensors and updates the total number of visitors on the LCD display.



💻 Features
Automatic visitor counting
Real-time display on LCD
Accurate entry and exit tracking
Simple and low-cost implementation
